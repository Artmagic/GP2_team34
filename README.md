Анализ благоприятных зон для открытия кафе в Москве

Команда: №34
12 Ноября 2025

Описание проекта

Цель проекта — определить наиболее благоприятные зоны для открытия кафе в Москве с учётом множества внешних факторов:
	•	уровня конкуренции,
	•	трафика пешеходов,
	•	стоимости аренды,
	•	инфраструктуры и транспортной доступности.

Ключевые вопросы исследования
	•	Где наблюдается низкая конкуренция между кафе?
	•	В каких районах фиксируется высокий поток людей?
	•	Где предлагается оптимальная аренда для помещений свободного назначения?
	•	Какие инфраструктурные элементы (метро, дороги, торговые центры) наиболее влияют на успех кафе?

Источники данных
	1.	CIAN — спарсено более 13 000 объявлений об аренде помещений в Москве.
Извлекались:
	•	название объявления,
	•	цена аренды,
	•	адрес и ближайшая станция метро,
	•	ссылка на оригинальное объявление.
	2.	Открытые городские данные Москвы — демографические и транспортные показатели.
	3.	Отзывные данные (например, с Яндекс.Карт или Google Maps) — количество отзывов и рейтинг заведений.

Методы анализа
	•	Корреляционный анализ между показателями проходимости, количеством отзывов, рейтингом кафе и инфраструктурой.
	•	Сегментирование Москвы по районам и факторам привлекательности.
	•	Построение индекса привлекательности, включающего параметры:
	•	арендная ставка,
	•	плотность кафе,
	•	индекс проходимости,
	•	транспортная доступность.
	•	Визуализация результатов с помощью тепловых карт и геоаналитики.

Основные результаты
	•	Подтверждена положительная корреляция между потоком людей и количеством отзывов (r = 0.57).
	•	Наибольшая зависимость наблюдается между:
	•	количеством отзывов и индексом проходимости (r = 0.611),
	•	средним рейтингом и длиной дорог (r = 0.601),
	•	количеством кафе и индексом проходимости (r = 0.569).
	•	Определены перспективные зоны для открытия новых кафе на основе рассчитанного индекса привлекательности.

Выводы
	•	Методика позволяет выявить районы с оптимальным соотношением арендной стоимости, потока людей и уровня конкуренции.
	•	Полученная модель может использоваться инвесторами и предпринимателями для выбора локаций под кафе и другие заведения общественного питания.
