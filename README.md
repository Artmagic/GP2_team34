Проект посвящён анализу привлекательности районов Москвы для открытия новых кафе.
Основная цель — определить, какие участки города имеют наилучший потенциал для успешного ведения бизнеса в сфере общественного питания.

Для этого была собрана и объединена информация из разных источников: данные о пассажиропотоке метро, средних арендных ставках, плотности кафе, рейтингах заведений, количестве инфраструктурных объектов и парковок.
Территория Москвы была разделена на равномерную сетку квадратов размером 800 на 800 метров, и для каждого квадрата рассчитаны ключевые показатели.

На основе этих данных был рассчитан интегральный индекс привлекательности.
Он отражает, насколько выгодно открывать кафе в конкретной точке города, учитывая поток людей, конкуренцию, инфраструктуру и стоимость аренды.

Основная идея формулы — чем выше пешеходная активность и качество инфраструктуры, тем привлекательнее место, но большое количество конкурентов и высокая аренда снижают итоговый индекс.
Веса факторов были определены экспериментально и основаны на анализе корреляций между признаками и популярностью заведений.

В результате получена карта Москвы с визуализацией распределения индекса привлекательности.
На тепловой карте хорошо видны районы с высокой коммерческой активностью — в первую очередь центр города и зоны с большим пассажиропотоком.
Проект демонстрирует, как данные из открытых источников могут быть использованы для анализа городской среды и поддержки управленческих решений.

Финальный датасет содержит все рассчитанные показатели и индекс привлекательности для каждого квадрата, что позволяет легко отобрать наиболее перспективные локации для открытия новых кафе.
